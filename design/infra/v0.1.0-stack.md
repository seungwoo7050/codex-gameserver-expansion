# v0.1.0 인프라 요약

## 구성 요소
- 서버: C++17 Boost.Beast 기반 바이너리, 8080 포트 노출
- Nginx: 80 포트에서 HTTP/WS를 서버로 리버스 프록시
- MariaDB: 3306 포트, 기본 DB `app_db`
- Redis: 6379 포트

## 네트워크/포트
- 외부 노출: 80(Nginx), 8080(서버 직접 접근 시)
- 내부 네트워크: `backend` 네트워크에서 컨테이너 간 통신

## 볼륨
- `mariadb_data`: MariaDB 데이터 영속화

## 환경변수 기본값
- SERVER_PORT=8080
- DB_HOST=mariadb / DB_PORT=3306 / DB_USER=app / DB_PASSWORD=app_pass / DB_NAME=app_db
- REDIS_HOST=redis / REDIS_PORT=6379
- LOG_LEVEL=info

## 기동/종료
- 기동: `cd infra && docker compose up --build`
- 종료: `docker compose down`
